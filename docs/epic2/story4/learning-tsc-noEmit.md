# 학습 — npx tsc --noEmit

---

## 각 부분의 의미

| 부분 | 의미 |
|---|---|
| `npx` | `node_modules/.bin/`에 설치된 CLI 도구를 실행한다. 전역 설치 없이 프로젝트에 설치된 버전을 사용할 수 있다. |
| `tsc` | TypeScript Compiler. `.ts` / `.tsx` 파일을 JavaScript로 변환한다. |
| `--noEmit` | 컴파일은 하되 JS 파일을 실제로 생성하지 않는다. 타입 오류 확인만 한다. |

---

## `--noEmit`이 필요한 이유

`tsc`를 그냥 실행하면 타입 검사 후 `dist/`에 JS 파일을 생성한다. order-web은 Vite가 빌드를 담당하므로 `tsc`가 JS 파일을 만들 필요가 없다. `--noEmit`으로 "타입 검사만 하고 파일은 건드리지 마라"고 지정한다.

---

## 언제 쓰는가

- 커밋 전 타입 오류 확인
- Task 완료 기준 검증 — 코드가 타입 오류 없이 컴파일되는지 확인
- Vite 개발 서버는 타입 오류가 있어도 실행되므로, 별도로 확인이 필요할 때
